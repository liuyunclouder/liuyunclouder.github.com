提升效率就是延长生命-MAC&iOS上的效率工具

Posted on 2017-03-25   |  

最近看到新闻http://daily.zhihu.com/story/9310547说workflow被苹果收购并免费了，之前就听说workflow是iOS平台上的效率神奇，于是第一时间下载试用。

一直都痴迷于试用工具和流程提升效率，无论是工作、学习还是娱乐：使用shell脚本一键发布、通过BetterTouchTool使用手势触发各种快捷键以及配合Keyboard Maestro设置的快捷键、一键下载Youtube上的视频播单，包括之前整理的个人信息系统，以及自己开发的Arsos和Afred的workflow以及Popclip的extension。

提升效率就是延长生命

每个效率工具都足以写一整篇博客来介绍，以下就列举下一些我常用的工具，排名不分先后

工作类

  • Tower：Git GUI工具
  • Git 配置
  • Droplr ：博客必备，已入终身会员
  • Popclip
  • RegExRX ：正则表达式神器
  • sip：屏幕取色
  • Paw：API 管理
  • BetterTouchTool：通过触摸板手势触发各种操作，非常强大
  • Keyboard Maestro：苹果官方Automator的升级版，为各种操作设置快捷键
  • Keycue：快捷键cheatsheet
  • SaneDesk：桌面文件分区工具，作者很强硬，就是不降价，忍痛入手
  • Navicat Premium：数据库管理工具，直观方便，UI漂亮
  • CodeRunner：集成各种编程环境，方便快速学习及测试编程语言
  • Kaleidoscope：漂亮的Diff工具
  • IconKit：做Appicon很方便
  • Charles & Wireshark：抓包利器
  • breaktime：番茄时钟
  • iTerm：oh-my-zsh、autojump必装，再配合上sshpass，妈妈再也不用担心我访问远程服务器了。再根据日常需要设置常用命令
  • starUML：画流程图比较方便
  • SnippetsLab：常用代码片段管理
  • Evernote：笔记神器
  • Alfred：各种workflow
  • Dash：各种编程语言及框架、数据库文档
  • moom：配合BetterTouchTool使用非常方便
  • path finder：系统Finder的升级版

学习类

  • Reeder：RSS聚合工具
  • MindNode Pro：画流程图必备
  • Ulysses：这篇博客就是用它写的
  • Timing：每日时间统计
  • Chrome extension：oneTab pocket 1password

娱乐类

  • Photodesk：Instagram浏览工具
  • Tweetbot：Tweet浏览工具
  • Fantastical：Todo+日历，人生必备啊
  • Boom2：Mac音效提升，听歌看电影效果很明显
  • 1Password：密码管理神器，再也不用记一大堆密码了

另外还有iOS上的IFTTT，平时用来看天气和球赛结果还是不错的。
