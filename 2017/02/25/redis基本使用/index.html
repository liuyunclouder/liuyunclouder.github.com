redis基本使用

Posted on 2017-02-25   |  

基本命令

安装

brew install redis

启动redis

redis-server

开机启动

brew services start redis

连接本地redis

redis-cli

连接远程redis

redis-cli -h {redis_host} -p {redis_port}

获取redis所有db的统计数据

info keyspace

进入db

select 2

获取当前db所有key

keys *

设置和获取

set varname value1 
get varname

清空

flushdb // 清除当前数据库的所有keys
flushall // 清除所有数据库的所有keys
